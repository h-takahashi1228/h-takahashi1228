## 🎨 WPF Level Editor

**Repository:** [リンク]  
**制作期間:** 約1ヶ月  
**担当:** 個人制作

WPF を用いて制作した、3D グリッドベースのレベルエディタです。

プランナーがレベルを構築し、そのデータを Houdini を経由して Unreal Engine へ渡すことを想定したツールパイプラインの理解・構築を目的に制作しました。

---

## 🎯 開発目的
- ツールエンジニア寄りのポートフォリオ制作
- WPF を用いた 3D ツール開発の基礎習得
- **WPF → Houdini → UE を想定したデータ連携設計の理解**
- レベル編集データを外部ツールで活用可能な形式で出力する設計
- Houdiniの基本的な使用方法と簡単なプロシージャル生成の習得

---

🛠 技術スタック
- C# / .NET
- WPF
- JSON（レベルデータ出力）
- Python（CSV 変換スクリプト）

---

## 🛠 実装した機能

### ■ レベル編集
- 3D Viewport 上でのブロック・オブジェクト配置 / 削除 / 編集
- 整数グリッドベースの座標管理
- カメラ移動・回転操作
- Undo / Redo 機能
- レベルデータの Import / Export（JSON）

### ■ データ連携
- JSON 形式でレベルデータを出力
- Python スクリプトで Houdini 用 CSV へ変換
- Houdini 上で CSV を読み込み、同等の 3D モデルを自動生成可能

---

## 🧠 設計面で意識した点

- レベルデータと描画処理の分離
- グリッド座標によるシンプルなデータ構造設計
- 外部ツール連携を前提とした拡張可能なデータ形式
- Undo / Redo を考慮した状態管理設計

## 🎮 SET – Solo Time Attack (Unreal Engine)

**Repository:** [リンク]  
**制作期間:** 約1ヶ月  
**担当:** 個人制作

ボードゲーム「SET」を題材にした、一人用タイムアタックゲームです。  
場にあるカードから正しい SET（3枚の組み合わせ）を見つけ続け、  
組み合わせが存在しなくなるまでの時間を競います。

---

## 🎯 開発目的
- UE5 を用いてミニゲームを一人で完成させる
- Blueprint によるロジック構築力の習得
- UI・演出・ゲーム進行管理を一通り実装する

2D でも実装可能な題材ですが、  
3D 空間上で構築することで UE の基本要素を総合的に理解することを目的としました。

---

### 🔍 ゲーム内容
- ランダムに生成されるカードから SET を探す  
- SET が存在しなくなった時点でゲーム終了  
- クリアタイムを記録  
- ローカルランキング機能でスコアを保存  
- 初回プレイ時に簡易チュートリアルを表示

---

## 🛠 技術スタック

- Unreal Engine 5
- Blueprint のみで全ロジック実装
- Python（カード画像生成）

---

## 🔍 実装内容

### ■ ゲームロジック

- SET 判定アルゴリズムを Blueprint で実装
  - 各カードの属性（形・色・塗り・数）を配列化
  - 「全て同じ」または「全て異なる」かを判定
- ランダムカード生成
- SET が存在しない場合のゲーム終了判定

### ■ UI / 演出

- カード選択時の強調表示
- カード再配置アニメーション
- タイマー表示
- 初回プレイ時チュートリアル
- ローカルランキング保存機能

### ■ アセット生成

- カード柄を Python スクリプトで自動生成
  - 作業効率化
  - デザインの一貫性維持
